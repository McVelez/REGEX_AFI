Convertidor de REGEX a AFI

    Consideraciones a seguir
  • Los únicos simbolos aceptados son las letras del alfabeto minúsculas y los numeros del 0 al 9.
  • Para operación de unión utilizar U mayúscula.
  • Para operación de estrella utilizar *.
  • Respetar el uso de paréntesis cuando sea necesario (En caso de realizar una operación estrella).
